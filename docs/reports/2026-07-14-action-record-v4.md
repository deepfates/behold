## What happened

One untasked inhabitant, `ActionWren`, entered a fresh Minecraft 1.21.4 epoch
and received the ordinary bounded player observation and seven admitted player
actions. An Ax mind using `google/gemini-3.5-flash` freely chose
`look_direction`.

The exact action passed through Behold's real `behold-default` permission
authority and action engine. Its terminal result and a later observation were
written into one private Lync life turn. A separate evaluator episode then
referenced that closed turn.

The result was projected into seven records:

1. observation before;
2. model proposal;
3. permission decision;
4. execution started;
5. execution completed;
6. observation after;
7. structural check.

The graph contains no `world_fact`. Looking is a real player action, but this
run did not earn a claim that Minecraft's material state changed.

## What this earns

We can now preserve the common causal shape of one inhabited turn without
making Minecraft's observations, permissions, physics, or facts generic. This
is a useful implementation inside Behold and a credible candidate for a shared
contract.

It has not yet earned a standalone package. That requires the same small graph
to survive a second real world without either adapter importing the other's
ontology.

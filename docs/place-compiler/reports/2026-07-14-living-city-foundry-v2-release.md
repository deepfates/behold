# Living City Foundry v2 release

Foundry v2 closes one reusable, place-data-driven path across San Francisco, lower Manhattan, and
Venice. The same compiler, runtime profiles, inspection, visit, ecology, performance, packaging,
and verification surfaces operate on all three. Place-specific geography and experience intent
remain declarative contracts; they are not branches in the release implementation.

The canonical evidence closure contains three structural inspections, three real 24,000-tick
ecology days, eighteen performance cases, three real-client visits (including one captured movie),
five capacity studies, three place-package indexes, and a portable Venice-to-Behold two-epoch
continuity proof. The release contract also carries unresolved frontiers so bounded successes
cannot be mistaken for whole-city geometry, ecology, rendering, or 2,048 concurrent LLM minds.

Run `npm run place:release-foundry -- --output <empty-directory>`, then independently run
`npm run proof:verify-foundry-release -- <directory>`. The packager verifies selected source
digests and builds a normalized archive. The verifier streams every archived member, checks its
embedded size and SHA-256, checks the outer checksum closure, and enforces the three-place user
story cardinalities without consulting the source workspace.

# Earth-to-Living-World bootstrap checkpoint

The first autonomous hero intent names Berkeley, describes the desired living-city experience, and
sets physical generation budgets. No place recipe, bounds, spawn, or landmarks were authored.

One real Nominatim request resolved Berkeley to OSM relation `2833528`. The deterministic budget
policy derived a 4.891 × 3.271 km, 16 km² world centered on the resolved place. The resolver response
is frozen with SHA-256 `a45325b3a3d1c715e3ac3cac30012dcb56549f02f9408d99713f06b963e9fc2d`.

One Overpass request then froze 728,039 OSM elements (80,155,967 bytes, SHA-256
`f6317e0579c42100e9dd8438d83b3fc17d89e0d86936215e27f02f3bbd484710`). The source contains 6,883
named elements and yielded 318 bounded landmark candidates. The first generic selection policy
collapsed into museum-like attractions and failed to preserve an open spawn. That failure motivated
a shared category-diversity and independent arrival policy. Replaying the same frozen source cost
zero provider calls and produced civic, education, landscape, vertical, transit, culture, district,
and everyday anchors with Ohlone Park as the leading physically unverified arrival candidate.

A real Ax 23.0.0 semantic call using `openai/gpt-4o-mini` consumed 7,196 tokens. It returned only
supplied IDs and sensible arrival candidates, but five of eight landmarks collapsed into the
culture family and its rationale referenced an unselected theater. The separate representation
gate rejected it. This is retained as evidence that typed generation is necessary but insufficient;
semantic output remains a proposal until judged, and all physical claims remain Minecraft's.

The compilation history is an authoritative Lync loom. Its eleven accepted events preserve the
intent, resolution, source acquisition, both deterministic proposals, the shared-policy revision,
the semantic call, and the corrective rejection without rewriting history. This loom records how a
world representation came to be. It does not replace Minecraft save/epoch history or Behold's
per-inhabitant memory.

# Compile and accept a third structurally distinct place

Select a geography that stresses different terrain, density, water, street, and landmark conditions; acquire a frozen input; generate, validate, inspect, soak, benchmark, repair, and package it through the same compiler.

## Acceptance Criteria

No place-specific implementation code; accepted immutable artifact and profiles; same route/sightline/survival/ecology/performance evidence; deterministic release; documented human interventions and remaining generality limits.

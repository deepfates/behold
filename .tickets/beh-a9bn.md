# Measure accelerated wide-world and controlled population frontier

Build causal capacity matrices that separately vary geographically separated active regions, tick acceleration, native entity counts, protocol bodies, and eventually Behold inhabitants/inference load.

## Acceptance Criteria

Real-server runners sweep declared region and load points with TPS/RSS/CPU/chunk/entity/liveness/error evidence, persistence and clean restart; unstable boundaries or honest lower bounds are identified; LOD/render reach is never presented as simulation reach; no linear thousands-agent claim.
